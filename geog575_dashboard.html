Aging America Dashboard

Video walkthough of the dashboard

In this project myself and another student created a dashboard helpful for identifying age trends over the past 50 years in Wisconsin counties. Users had the ability to view demographic information about each county between 1970 - 2020, and could pin population pyramids beside each other to compare. Users can also toggle layers with information about current healthcare offerings, which can help plan for potential expansion of services in certain counties as populations continue to grow old

Technologies Used

-- JavaScript
-- Leaflet
-- D3.js
