Full Stack Weather Application

Video walkthrough of the application

This project combined frontend and backend coding to create a full stack application. The backend was hosted on an AWS EC2 instance, with API calls being made to the free OpenWeatherMap service. The front end was styled using Bootstrap, and used Leaflet to visualize the API requests. The background of the application dynamically updates based on the current weather of the searched area. Four APIs from the OpenWeatherMap service were used to fully create this project.
